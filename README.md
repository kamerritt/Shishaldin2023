# Shishaldin2023
Code to compare infrasound RMS Pressure (Pa) with volcanic radiative power (W) from Events 3, 6, and 12 of the 2023 eruption sequence of Shishaldin Volcano, Alaska, USA. Script will output a plot of infrasound and seismic amplitudes, volcanic radiative power (VRP), explosion triggers, and ash plume height.
